Vectors Operations
1. v.capacity() :- Tells the Size of the vector
2. v.size() :- No. of elements in the vector
3. v.push_back(value) :- add an element to vector
4. v.at(index) :- access element at any index
5. v.pop_back() :- delete last element
6. v.front() :- points to first element
7. v.back() :- points to last element
8. v.clear() :- deletes all elements in a vector
9. v.resize(size) :- Resize vector to store n elements
10. v.shrink_to_fit() :- Resize according to number of elements present in the vector
11. v.empty() :- CHeck whether vector is empty or not
12. v.insert(position, value) :- Insert a value at a given position
13. v.erase(position) :- Delete an element at a given Position
